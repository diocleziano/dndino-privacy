# Overview

DnDino helps the Dungeon Master manage:

- characters
- adventures
- places
- place occupants
- combat
- the live session for players
- support material

## Where to Start

If you want to get comfortable with the app without feeling lost, this is the recommended path:

1. read the general overview
2. learn how to create a character
3. create your first adventure
4. take a look at the adventure dashboard
5. link your heroes to the adventure
6. add a few images
7. create places or quests inside the adventure
8. add NPCs or monsters to a place
9. create a test combat using the heroes and the NPCs or monsters from that place
10. run a short test combat
11. as the adventure grows, start using concept maps and interactive maps to organize the campaign more clearly

!!! tip
    You don't need to configure everything at once. DnDino works best when you build the campaign step by step: first the characters, then the adventure, then the places, and finally the combat flow.
